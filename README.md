EgcSimu Vue3

By b站水杨酸酸

仿《蛋仔派对》工坊蛋码编辑器，网页前端实现。
使用Vue3+TS重构的蛋码模拟器，可以在浏览器中高效运行，提供便捷的蛋码编辑功能。
界面比原始版本更美观合理，因为我在做以前那个的时候刚学原生前端。
本项目易于维护（TS），欢迎贡献代码。

立刻使用

访问https://eggyhub.top/egcs立刻使用本编辑器，无需下载部署。

启动方式
安装Node.js
npm install
npm run dev（本地立刻启用）
访问http://localhost:5173

npm run build（打包项目）
在dist目录下生成打包后的前端文件。

使用说明

你可以根据蛋码编辑器的使用方法来进行拼蛋码，保存，撤回等操作。
如果发现部分类型错误，是正常现象，可反馈。
本项目目前只用于展示或教学演示功能，暂无法转换为可用蛋码。
